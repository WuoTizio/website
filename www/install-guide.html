Install Guide

In order to install Jolie successfully, follow these instructions:

  1. Run the Shell Terminal as ADMINISTRATOR
  2. Change your path corresponding where you have downloaded the Jolie Installer
  3. Execute the command "java -jar jolie-1.6.2.jar"

    - on Linux/MacOS, by executing the installer as root, write the command "sudo java -jar jolie-1.6.2.jar"

  4. Follow the displayed instructions

Great! Now you're ready to enjoy Jolie:
